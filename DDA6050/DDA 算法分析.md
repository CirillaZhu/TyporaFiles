# DDA 算法分析

#### 主定理

| $T(n) <= a·T(n/b) + O(n^d)$ | if      | =                    |
| --------------------------- | ------- | -------------------- |
| $a >= 1,b > 1, c>=0$        | $a=b^d$ | $\Theta(n^dlogn)$    |
|                             | $a<b^d$ | $\Theta(n^d)$        |
|                             | $a>b^d$ | $\Theta(n^{log_ba})$ |
